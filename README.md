## 🌟 Overview

Psychic System is a web-based chess application designed to be fast, accessible, and easy to use. Built entirely without heavy frameworks or backend dependencies, it offers a seamless experience for playing chess, solving tactical puzzles, and studying the game.

## ✨ Features

- **♟️ Play Chess**: A fully functional chess board with complete rule validation.
- **🧩 Chess Puzzles**: Sharpen your tactical vision with a dedicated puzzle section.
- **📚 Learn & Openings**: Built-in resources to study chess fundamentals and explore popular opening theory.
- **⚡ Lightweight & Fast**: Built with vanilla web technologies (HTML, CSS, JS) for lightning-fast load times.
- **📱 Responsive Design**: Optimized for both desktop and mobile browsers.

## 🚀 Getting Started

Since this is a purely static web project, you can run it locally without any complex build steps, Node.js, or package managers.

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge).
- *(Optional)* A local development server (like the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension for VS Code) is recommended to avoid any potential browser security/CORS issues.

### Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/simpler-code/psychic-system.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd psychic-system
   ```

3. **Launch the game:**
   - Simply open `index.html` in your web browser.
   - Or, if using a local server extension, start the server and navigate to `http://localhost:<port>`.
